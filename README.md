# Dormez
Dormez is an interpreted programming language. It supports the following:
* Primitive types (number, string, bool, etc.)
* Non-primitive types (console, file, etc.)
* If/else if/else statements
* All kinds of loops
* Functions
* Lua-style tables
* Object-oriented programming
